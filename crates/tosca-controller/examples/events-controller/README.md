# `events-controller`

[![LICENSE][license badge]][license]

A controller that scans the network for `tosca` devices and subscribes to
their brokers to receive events.

This example demonstrates how to build a **Text User Interface (TUI)** that 
utilizes the event management capabilities of the `tosca-controller`.

## Usage

To build the example:

```console
cargo build
```

To run the example:

```console
cargo run
```
