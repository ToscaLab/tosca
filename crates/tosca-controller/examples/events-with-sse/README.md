# `events-with-sse`

[![LICENSE][license badge]][license]

A controller that scans the network for `tosca` devices, subscribes to their
brokers to receive events, and displays their data in real-time on a web page.

This example demonstrates how to build a **Web Application** that utilizes the
event management capabilities of the `tosca-controller`.

## Usage

To build the example:

```console
cargo build
```

To run the example:

```console
cargo run
```
